# Credit Card Fraud Detection using Machine Learning

This project compares Decision Tree and K-Nearest Neighbors (KNN) algorithms for detecting fraudulent credit card transactions using a real-world dataset. The goal is to evaluate both models based on classification performance and computational efficiency.

**Features**
- Data preprocessing
- Feature scaling
- Model training
- Performance evaluation
- Algorithm comparison
- Visualization
- Execution time analysis

**Dataset**
Source: Kaggle
Number of transactions (568,630)
31 attributes
Binary classification (Fraud / Legitimate)

**Technologies**
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

**Algorithms**
- Decision Tree
- K-Nearest Neighbors (KNN)

**Evaluation Metrics**
- Accuracy
- Precision
- Recall
- F1-score
- Running Time
